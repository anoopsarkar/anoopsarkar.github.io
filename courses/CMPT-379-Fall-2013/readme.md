Setup
-----

* download bootstrap (http://getbootstrap.com/)
* copy dist/css/bootstrap.min.css to css/
* copy dist/js/bootstrap.min.js to js/
* curl http://code.jquery.com/jquery.js > js/jquery.js
* copy dist/fonts/*.* fonts/
